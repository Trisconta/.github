### Excercising dzdata using `load_dbz.py`
The following .idea/workspace.xml are options/ examples for `load_dbz.py`:
```
<option name="PARAMETERS" value="--filter A --filter S" />
<option name="PARAMETERS" value="--filter A:658386921,356988887" />
<option name="PARAMETERS" value="--filter A:302792,1347441 --filter T:3149589 --filter S:7068771" />
```
